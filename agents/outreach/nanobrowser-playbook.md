# Outreach Agent — Nanobrowser Playbook

## Fiverr Buyer Requests
1. Open Nanobrowser in Chrome
2. Command: "Go to fiverr.com/buyer_requests and list all requests 
   for web design, 3D animation, or logo design posted today. 
   Include: title, description, budget, deadline."
3. Copy results → paste into Open WebUI Outreach Agent
4. Outreach Agent writes custom proposals
5. Manually submit proposals on Fiverr

## Upwork Job Board Sweep
1. Command: "Search upwork.com for jobs: web design, website development, 3D animation. 
   Filter: posted last 24h, budget $500+. 
   Extract: title, description, client history, budget."
2. Pipe results to Open WebUI Outreach Agent for proposal generation
3. Proposals saved to ~/Desktop/OPENCLAW/outputs/proposals/

## Facebook Business Group Outreach
1. Command: "Search Facebook groups for 'local business owners [your city]', 
   'small business networking'. Find posts where people mention needing a website."
2. Outreach Agent generates friendly DM scripts
3. Manually send via Facebook

## Cold Email Lead Gen
1. Command: "Search Google for '[city] restaurants without websites', 
   '[city] small businesses'. Find contact emails."
2. Coder Agent writes Python email script
3. Outreach Agent writes personalized email template
4. n8n sends emails via SMTP
