## 4. Gaps & Recommendations

1. **Supabase not wired into lead storage**: `lib/supabase.ts` exists but `storeLead()` doesn't use it. Wiring Supabase as the primary durable store (before Google Sheets) would make leads survive Vercel deployments on the free tier (SQLite is ephemeral).

2. **npm vulns**: `postcss` has a direct patch. Run `npm update postcss` for the moderate XSS. Next.js 14 EOL is approaching; plan a React 19 + Next.js 15 migration path.

3. **Contact form**: `/app/contact/page.tsx` is modified but not yet verified in this cycle. The contact API route exists (`/api/contact`).

4. **No UI tests**: E2E or component tests would catch regressions in the audit funnel flow.
