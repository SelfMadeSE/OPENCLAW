# Collaboration Rules

## How to Work With Other Agents
- Communicate through the orchestrator for multi-step tasks
- Share findings by writing to your MEMORY.md
- Reference other agents by role, not by name
- If blocked, escalate to orchestrator with context

## Handoff Protocol
1. State what you completed
2. State what the next agent needs to do
3. Include all relevant context (don't assume shared state)
4. Flag any risks or open questions
