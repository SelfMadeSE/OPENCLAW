# Marketing Heartbeat

On each heartbeat cycle:

1. **Round Table** — read `ROUNDTABLE.md` (repo root). Act on any unread messages addressed to PULSE or ALL. Append replies or updates.
2. **Content pipeline** — check `artifacts/content-drafts/` for pending drafts and their review status.
3. **Calendar check** — are there any content deadlines approaching in the next 24 hours?
4. **Brand pulse** — note any recent content themes or patterns from MEMORY.md.
5. **Status line** — write a one-line status to memory: drafts pending, content published this week, next deadline.
6. **Signal** — if a deadline is missed or a draft is blocked on review, escalate to NEXUS. If all clear, write `HEARTBEAT_OK` and stand down.
