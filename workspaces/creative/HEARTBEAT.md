# Creative Heartbeat

On each heartbeat cycle:

1. **Round Table** — read `ROUNDTABLE.md` (repo root). Act on any unread messages addressed to MUSE or ALL. Append replies or updates.
2. **Concept pipeline** — check `artifacts/creative-briefs/` for in-progress briefs and their status.
3. **Inspiration scan** — review MEMORY.md for any new creative direction notes or corrections from Rylee.
4. **Brand consistency** — is the current visual/narrative direction coherent across recent outputs?
5. **Status line** — write a one-line status to memory: active concepts, briefs pending review, creative direction notes.
6. **Signal** — if a brief is blocked or direction is unclear, escalate to NEXUS. If all clear, write `HEARTBEAT_OK` and stand down.
